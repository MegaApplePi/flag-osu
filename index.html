Help Example How to use flag-osu

  1. Start with the tournament article you need flag reference links for.

  2. Before you begin using flag-osu, you need to make sure that the article you are using uses the correct flag reference names that flag-osu looks for.
    The format looks like this:

    • ![][flag_XX], where XX is the ISO 3166-1 alpha-2 country codes in uppercase letters.

    • ![][flag_XXYY], where XXYY is a combination of the ISO 3166-1 alpha-2 country codes in uppercase letters (you will rarely use this at all).

    • You can also see ppy/osu-wiki/issues/328 for a full list of osu-wiki flags icons.

  3. Copy and paste the tournament article into the input text field (left).

  4. Click the Find Flags button. This will parse the text and will return an output on the right.

  5. Inspect the output text field (right).

    • If the output appears OK, copy the output text field (right) and paste it back into the article.

    • If not, check the input text field (left) or open an issue (or ping MegaApple_Pi on the osu!dev Discord).
